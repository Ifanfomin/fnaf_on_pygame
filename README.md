# FNAF on pygame
### Launch game:
Install requirements:
```commandline
pip install -r requirements.txt
```
Start main.py:
```commandline
python main.py
```

### Controls:
- Enter - start game
- Escape - exit game
- D - go left
- J - go right
- A - close right door
- L - close left door
- Space - on/off light
- Shift - open/close cameras
- 1, 2, 3, 4, 5, 6 - camera number
